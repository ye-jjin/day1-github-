### wsl란?
- WSL란 Windows Subsystem for Linux의 약자로, 윈도우 10에서 리눅스 환경을 구축하기 위한 가상화 기술입니다. 
- WSL을 사용하면 Windows 운영체제에서 리눅스 명령어를 실행할 수 있으며, 리눅스 응용 프로그램을 설치하고 실행할 수 있습니다.
